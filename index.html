MASAR AI Course Syllabus

A course covering Language Models, Transformers, Large Language Models (LLMs), and Retrieval Augmented Generation (RAG).

Course Content

Week Main Topics Key Concepts Assignments & Practical Work
Week 1 - Intro to Language Models
- Traditional LMs (N-grams)
- Recurrent Neural Networks (RNNs)
- Introduction to Transformers
- Pre-training & Fine-tuning
- Speech Recognition (ASR)
  • N-gram Language Modeling
  • Perplexity Evaluation
  • RNNs, LSTMs, and Seq2Seq
  • Vanishing/Exploding Gradients
  • Seq2Seq with Attention
  • Self-Attention, Q, K, V
  • BERT (MLM, NSP)
  • GPT (Causal LM)
  • ASR, MFCC, CTC Loss, Wav2Vec
  • Homework 1: Fine-tuning wav2vec 2.0 for ASR on Arabic Common Voice.
Week 2 - Transformers Recap
- Intro to Large Language Models (LLMs)
- Memory Analysis (Training & Inference)
- Parameter-Efficient Fine-Tuning (PEFT)
- LLM Evaluation & Serving
- In-Context Learning & Prompting
  • Decoder-Only Architecture (GPT, Llama)
  • Softmax with Temperature
  • Sampling: Greedy, Top-K, Top-P
  • Estimating Memory Requirements
  • LoRA (Low-Rank Adaptation)
  • QLoRA (Quantized LoRA)
  • Standard Benchmarks
  • Prompt Engineering
  • Zero-shot, One-shot, Few-shot Learning
  • Homework 2: In-context learning for sentiment analysis using llama 3.1 8B.
Week 3 - Similarity & Distance Metrics
- Embedding Models
- Contrastive Learning
- Query-Document Scoring
- Semantic Search & Document Retrieval
- RAG Pipelines
  • Cosine Similarity vs. Dot Product
  • L1 & L2 Distance
  • Sentence Embedding Models
  • Contrastive Learning (Triplet loss, infoNCE)
  • Shared Encoder vs. Bi-Encoder
  • Cross-Encoder (Reranker)
  • Vector Stores
  • RAG Pipeline: Retrieve, Rerank, Generate
  • Reducing Hallucination with RAG
  • Hands on Metrics
  • Query-Document scoring hands on
  • Search Engine
  • Fully functional RAG pipeline
